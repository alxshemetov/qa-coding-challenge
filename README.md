# qa-coding-challenge

## Tech stack:

1. Java 11
2. Maven
3. JUnit 5
4. REST-assured
5. Allure Framework

## Run:

`mvn clean test`

## Install Allure Framework:

[Installing a commandline](https://docs.qameta.io/allure/#_installing_a_commandline)

## Tests results:

`$ allure serve target/allure-results`
